# Vue Learn

> A Vue.js2.0 project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

## Dev Environment

npm run dev

访问地址：
<ul>
<li>http://localhost:9091/module/index.html</li>
<li>http://localhost:9091/module/basedemo.html</li>
</ul>
